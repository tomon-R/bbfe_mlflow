\section{3次元スロッシング}

\subsection{解析条件}

水平加速度$f_{x}$は以下の式で与える。
\begin{equation}
	f_{x} = A \omega^2 \sin{\omega t}
\end{equation}
ここで、振幅$A=0.0093 \mathrm{m}$、角速度$\omega = 5.311 \mathrm{rad/s}$である。
